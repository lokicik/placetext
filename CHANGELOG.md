## [0.1.0] - 2025-11-14

### Added

- Initial release of placetext
- Zero-dependency placeholder text generator
- TypeScript implementation with full type definitions
- Deterministic PRNG (mulberry32) for seeded generation
- Five built-in corpora:
  - Classic (lorem ipsum style)
  - Tech (programming terminology)
  - Startup (business buzzwords)
  - Concise (common English words)
  - Musk (characteristic phrases)
- Custom corpus support
- Generate words, sentences, or paragraphs
- Configurable sentence structure and paragraph formatting
- ESM and CommonJS builds
- Node.js 18+ and browser support
- Comprehensive test suite
- Professional documentation
